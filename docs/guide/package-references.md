# Package References

This page catalogs the third-party packages used across the app, grouped by purpose (core framework and mapping). It is important because it gives contributors and maintainers a quick dependency reference for setup, troubleshooting, security reviews, and upgrade planning.

## Package List

### Base Framework

- [Axios](https://github.com/axios/axios)
- [destr](https://github.com/unjs/destr)
- [iconify](https://github.com/iconify/icon-sets)
- [Nuxt](https://github.com/nuxt/nuxt)
- [Nuxt Auth Utils](https://github.com/atinux/nuxt-auth-utils)
- [Nuxt UI](https://github.com/nuxt/ui)
- [Pinia](https://github.com/vuejs/pinia)
- [pinia-persisted-state](https://codeberg.org/praz/pinia-plugin-persistedstate/)
- [TypeScript](https://github.com/microsoft/TypeScript)
- [vue](https://github.com/vuejs/core)
- [vue-router](https://github.com/vuejs/router)
- [zod](https://github.com/colinhacks/zod)

### Mapping

- [MaxMind GeoIP2 Node.js API](https://github.com/maxmind/GeoIP2-node)
